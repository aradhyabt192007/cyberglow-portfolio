

## Cybersigilism Photography Portfolio

A single-page, darkly immersive portfolio with a "Neo-Tribal Underground" aesthetic — sharp lines, glitch effects, and dramatic reveals.

### Sections & Features

1. **Hero / Landing**
   - Fullscreen image with heavy grain overlay
   - Large animated sigil in the center that fades on scroll
   - Your name in Syne Extra Bold, metadata in JetBrains Mono
   - Scroll indicator with a pulsing thorn-like arrow

2. **About Me**
   - Asymmetric layout with a portrait slot and bio text
   - Thin silver sigil lines framing the section
   - Technical metadata styling (location, gear, style)

3. **Photo Gallery**
   - Masonry grid with varying image sizes
   - Chromatic aberration flicker effect on hover
   - Thin silver connecting lines between images
   - Category filter chips (Street, Portrait, Abstract, etc.) with glitch transitions
   - Lightbox on click for fullscreen viewing

4. **Contact**
   - Minimal contact form with sharp edges (no border-radius)
   - Social links with Electric Crimson hover states
   - Email displayed in JetBrains Mono

### Design System
- **Background:** #050505 (Deep Obsidian)
- **Lines/Sigils:** #E5E7EB (Liquid Silver)
- **Accent:** #FF003C (Electric Crimson) — hover states, active elements
- **Text:** #F3F4F6 (Ghost White)
- **Fonts:** Syne (headings), JetBrains Mono (body/metadata)
- **No rounded corners, no soft shadows**

### Interactions
- Custom cursor with faint thorn trail
- Scroll-triggered sigil line animations in background
- Near-instant, glitchy transitions throughout
- Image flicker reveal on hover with chromatic aberration
- Navigation using abstract line clusters instead of hamburger menu

### Content
- 8-10 sample placeholder images from Unsplash (photography themed)
- Placeholder name and bio text
